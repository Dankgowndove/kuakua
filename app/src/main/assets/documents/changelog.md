# 更新日志

## 版本 1.1.0

### 新增功能

- 自定义背景图片：支持从相册选择图片作为背景
- 词库导入导出：支持将词库导出为 JSON 文件或从文件导入
- 文本编辑器：带行号、统计、撤销/重做的纯文本编辑工具

### 修复

- 修复渐变背景选择逻辑错误
- 修复分类索引越界风险
- 替换已废弃的 API 调用
- 清理无效代码和无用依赖

## 版本 1.0.0 (2025-12-27)

### 新增功能

- 核心功能：点击按钮获得随机夸赞
- 设置页面：长按按钮进入设置
- 背景音乐：支持开关、切换和音量控制
- 背景设置：支持纯色和渐变背景
- 主题设置：支持多种主题和颜色方案
- 词库管理：支持添加、编辑和删除自定义夸赞
- 文档系统：内置使用说明、架构文档等

### 技术特性

- MVVM 架构
- Jetpack Compose UI
- Material Design 3 设计
- DataStore 数据持久化
- Room 数据库
- ExoPlayer 音乐播放
- Navigation Compose 导航

### 计划功能

- 更多背景音乐
