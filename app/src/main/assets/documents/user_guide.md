# 使用说明

## 基本功能

### 获得夸赞

1. 打开应用后，点击屏幕中央的"夸我一下"按钮
2. 应用会随机显示一条夸赞语句
3. 夸赞语句会以弹窗形式出现在屏幕上方
4. 继续点击按钮可以获得更多夸赞

### 进入设置页面

点击右上角的设置图标即可进入设置页面。

## 设置功能

### 背景音乐

- **开关**：开启或关闭背景音乐
- **音乐切换**：选择不同的背景音乐
- **音量控制**：调整背景音乐的音量大小

### 背景设置

- **纯色背景**：从预设颜色中选择纯色背景
- **渐变背景**：从预设渐变方案中选择渐变背景
- **自定义背景**：从相册选择图片作为背景

### 主题设置

- **主题模式**：选择默认、暗色或亮色主题
- **主题颜色**：选择不同的主题颜色方案

### 词库管理

- **查看统计**：查看内置语句、自定义语句和总数
- **添加语句**：添加自定义夸赞语句
- **编辑语句**：编辑自定义语句的内容和类别
- **删除语句**：删除不需要的自定义语句
- **导出/导入**：支持将词库导出为 JSON 文件或从文件导入

### 文档

查看详细的使用说明、系统架构、开发文档等。

### 工具箱

- **文本编辑器**：带行号、统计、撤销/重做的纯文本编辑工具

## 常见问题

### 为什么音乐不播放？

请检查：
1. 设置页面中音乐开关是否开启
2. 设备音量是否已打开
3. 应用是否有音频播放权限

### 如何恢复默认设置？

目前需要手动调整各项设置到默认状态。

### 自定义语句会丢失吗？

不会，自定义语句保存在本地数据库中，不会丢失。
